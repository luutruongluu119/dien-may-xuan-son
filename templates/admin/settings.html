Cách lấy Page ID: vào trang Facebook của shop → "Giới thiệu" (hoặc Cài đặt trang → Thông tin trang) → tìm dòng "Mã số trang"/"Page ID" (dãy số) → dán vào ô trên. Sau đó vào Cài đặt trang → Nhắn tin → bật "Tiện ích trò chuyện" (Chat plugin) và thêm domain {{ request.host }} vào danh sách miền được whitelist thì khung chat mới hiện được trên web. Để trống thì web không hiện khung Messenger (không lỗi gì cả).

Facebook Pixel (chạy quảng cáo mục tiêu Chuyển đổi)

Pixel giúp Facebook học ai là khách thật từ web này (xem sản phẩm, để lại thông tin đặt hàng), để Xưởng Quảng Cáo chạy được mục tiêu Chuyển đổi thay vì chỉ Tin nhắn.

Cách tạo: vào business.facebook.com/events_manager2 → "Kết nối nguồn dữ liệu" (Connect Data Sources) → chọn "Web" → "Pixel Facebook" → đặt tên (VD "Điện Máy Xuân Sơn — Website") → nhập {{ request.host }} → Tạo. Copy dãy số Pixel ID hiện ở trang tổng quan, dán vào ô trên.

Giúp web tự báo cho Facebook biết ai vừa để lại thông tin đặt hàng (sự kiện "Lead") thẳng từ server — đáng tin hơn nhiều so với chỉ báo qua trình duyệt (không bị AdBlock chặn mất). Cách lấy: trong Events Manager, chọn đúng Pixel vừa tạo ở trên → tab "Cài đặt" (Settings) → kéo xuống "Conversions API" → "Đặt tạo access token thủ công" (Generate access token) → copy chuỗi hiện ra (chỉ hiện 1 lần) → dán vào ô trên. Để trống thì web vẫn chạy bình thường, chỉ là không có lớp báo dự phòng này (Pixel phía trình duyệt vẫn hoạt động).

Cấu hình AI viết bài

Cả 2 key bên dưới đều bật khung chat AI tư vấn cho khách nổi trên web (chỉ cần 1 trong 2 là đủ, để trống cả hai thì khung chat không hiện) — nếu có cả hai, hệ thống ưu tiên dùng Gemini trước vì Gemini có gói miễn phí, Claude API là trả phí theo lượng dùng.

Gemini API key — miễn phí, lấy tại aistudio.google.com/apikey. Dùng cho chat AI tư vấn khách và tự tạo ảnh minh hoạ cho bài viết.

{% if env_gemini %}

✅ Đang dùng Gemini key cấu hình an toàn qua biến môi trường Vercel (không lưu trong file này để tránh lộ trên GitHub công khai). Muốn đổi key thì vào Vercel → Settings → Environment Variables → GEMINI_API_KEY. Ô bên dưới chỉ áp dụng khi chạy ở máy local, không ảnh hưởng bản đang chạy thật.

{% endif %}

Claude API key — trả phí theo lượng dùng, dùng để bấm "Tạo nháp bằng AI" khi viết bài trong mục Tin tức, và cũng có thể dùng cho chat AI nếu bạn không dùng Gemini. Lấy key tại console.anthropic.com.

{% if env_claude %}

✅ Đang dùng Claude key cấu hình qua biến môi trường Vercel (CLAUDE_API_KEY), tương tự Gemini ở trên.

{% endif %}

Lịch tự động: khi bật, hệ thống tự lấy chủ đề từ Hàng đợi để viết nháp + tạo ảnh theo tuần — bài luôn ở dạng nháp chờ bạn duyệt, không tự đăng thẳng. Chỉ chạy khi app chạy như 1 tiến trình sống lâu dài (VPS), không có tác dụng trên bản Vercel hiện tại.
